# Color Palette Suggestion for MountLift Website

## Recommended Palette: "Vibrant Professional"

A modern, eye-catching color scheme that balances professionalism with creative energy - perfect for an influencer marketing agency.

### Primary Colors

**Deep Indigo (Primary Brand Color)**
- Main: `#6366F1` (Indigo-500)
- Dark: `#4F46E5` (Indigo-600)
- Light: `#818CF8` (Indigo-400)
- Usage: Primary buttons, CTAs, brand highlights, navigation accents

**Vibrant Coral (Secondary/Accent)**
- Main: `#F97316` (Orange-500)
- Dark: `#EA580C` (Orange-600)
- Light: `#FB923C` (Orange-400)
- Usage: Secondary CTAs, hover states, feature highlights, tool icons

**Electric Cyan (Tertiary/Accent)**
- Main: `#06B6D4` (Cyan-500)
- Dark: `#0891B2` (Cyan-600)
- Light: `#22D3EE` (Cyan-400)
- Usage: Accent elements, links, interactive states, decorative elements

### Supporting Colors

**Rich Purple (Premium/Exclusive)**
- Main: `#8B5CF6` (Violet-500)
- Usage: Exclusive representation section, premium features

**Vibrant Pink (Creative/Energy)**
- Main: `#EC4899` (Pink-500)
- Usage: Social media elements, creative tools, highlights

**Success Green**
- Main: `#10B981` (Emerald-500)
- Usage: Success states, positive metrics, growth indicators

### Neutral Palette (Enhanced)

**Backgrounds**
- Primary: `#FFFFFF` (White)
- Secondary: `#F8FAFC` (Slate-50) - slightly warmer than gray-50
- Tertiary: `#F1F5F9` (Slate-100)

**Text**
- Primary: `#0F172A` (Slate-900) - deep, rich black
- Secondary: `#475569` (Slate-600) - readable gray
- Tertiary: `#94A3B8` (Slate-400) - subtle text

**Borders & Dividers**
- Light: `#E2E8F0` (Slate-200)
- Medium: `#CBD5E1` (Slate-300)

### Gradient Combinations

1. **Hero Gradient**: `from-indigo-600 via-purple-600 to-pink-500`
2. **CTA Gradient**: `from-indigo-600 to-violet-600`
3. **Card Hover**: `from-indigo-50 to-purple-50`
4. **Background Accent**: `from-indigo-100/20 via-purple-100/20 to-pink-100/20`

## Implementation Recommendations

### Where to Apply Colors

1. **Primary CTA Buttons**: Use Indigo-600 (`#4F46E5`) with white text
2. **Hover States**: Transition to Indigo-700 or add Coral accent
3. **Navigation**: Indigo for active states, Coral for hover
4. **Section Headers**: Gradient text using Indigo → Purple → Pink
5. **Icon Backgrounds**: Use the vibrant colors (Coral, Cyan, Purple, Pink) for different tool categories
6. **Cards**: Subtle Indigo-50 background on hover
7. **Links**: Cyan-500 for primary links, Indigo-600 for secondary
8. **Footer**: Keep black background but add Indigo accent to logo/links

### Color Psychology

- **Indigo**: Trust, professionalism, creativity, innovation
- **Coral/Orange**: Energy, enthusiasm, creativity, action
- **Cyan**: Freshness, clarity, communication, modern
- **Purple**: Premium, exclusive, creative, luxury
- **Pink**: Social, energetic, creative, youthful

## Accessibility Notes

- All color combinations meet WCAG AA contrast standards
- Indigo-600 on white: 7.1:1 contrast ratio ✅
- White text on Indigo-600: 7.1:1 contrast ratio ✅
- Ensure hover states maintain sufficient contrast
